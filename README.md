## Terraform
#### Khi clone về cần vào thư mục terraform thực hiện điền các value cho arguments sau đó thực hiện các bước sau:
```bash
# 1. Copy file example thành file thật
cp terraform.tfvars.example terraform.tfvars

# 2. Mở file và điền giá trị của mình vào
# Sau đó chạy bình thường
terraform init
terraform plan
```
